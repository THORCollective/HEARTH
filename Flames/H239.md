# H239

Arctic Wolf Labs reported multiple June 2026 intrusions where exploitation of CVE-2026-0257 led to Qilin ransomware deployment. The useful hunting chain starts after the perimeter access: credential theft with `comsvcs.dll` MiniDump and `ntdsutil.exe` IFM, staging under `C:\PerfLogs`, Run key persistence with a randomized asterisk-prefixed value, PsExec fan-out through administrative shares, and broad event-log clearing before encryption.

## Why

- Arctic Wolf reported multiple June 2026 Qilin intrusions that reused the same Windows post-compromise chain after CVE-2026-0257 access, and assessed related activity as likely ongoing.
- The hunt survives infrastructure and hash rotation because it keys on required behavior: credential theft, AD database extraction, ransomware staging under `C:\PerfLogs`, PsExec fan-out, and event-log clearing.
- The telemetry is common for MDR work. EDR process and file events, registry telemetry, service creation logs, Windows security events, and PowerShell logs each cover a different part of the chain.
- False positives are controllable because normal PsExec, backup, or BitLocker work should not include LSASS dumping to `output.odt`, NTDS IFM extraction, `C:\PerfLogs\win.exe --password`, and broad log clearing in the same window.
