# H-2025-001

Threat actors are using PowerShell scripts such as slr.ps1, sqlx.ps1, sav.ps1 and cfg.ps1 to extract Classroom Spy files from ZIP archives and install the software as a service for remote administration of compromised systems.

## Why
- Classroom Spy provides attackers with extensive remote access capabilities on compromised systems, including keylogging, screen monitoring, file collection/deployment, and more
- Installing Classroom Spy as a service allows it to persist across system reboots
- This activity is part of a larger campaign targeting financial institutions in Africa, likely to establish initial access footholds that can be sold to other threat actors
