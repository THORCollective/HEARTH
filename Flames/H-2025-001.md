# H-2025-001

Threat actors are using the remote administration tool Classroom Spy to monitor victim computers, collect files, log keystrokes, and execute commands on compromised hosts in the financial sector.

## Why
- Classroom Spy provides attackers with extensive remote access capabilities on victim machines, allowing them to surveil user activity, exfiltrate sensitive data, and run arbitrary commands. This level of control is alarming, especially in the financial sector.
- By deploying Classroom Spy, the threat actors can maintain persistent access to compromised networks in financial institutions. This allows them to conduct espionage, commit fraud, and potentially sell this access to other criminal groups on the dark web.
- The abuse of Classroom Spy appears to be part of a larger campaign by an initial access broker tracked as CL-CRI-1014 that is targeting financial organizations across Africa. Detecting this tool is important for identifying intrusions by this threat actor.
