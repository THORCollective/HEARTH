# H-2025-001

CTI REPORT:

Our intelligence sources have identified a new campaign by an unknown threat group targeting financial institutions. The attackers are using spear-phishing emails to deliver a malicious Word document. Once the document is opened, it exploits a vulnerability in Microsoft Office to execute a PowerShell script that downloads a second-stage payload from a remote server. The payload is a custom backdoor that uses Windows Management Instrumentation (WMI) to execute commands and exfiltrate data.

---

Hypothesis:

Threat actors are exploiting a vulnerability in Microsoft Office to execute a PowerShell script that downloads a malicious payload from a remote server, with the immediate tactical goal of installing a custom backdoor on systems within financial institutions.

## Why
- Detecting this behavior is crucial as it represents the initial access vector used by the threat actors, allowing them to gain a foothold in the target network.
- If successful, the attackers can establish persistence, move laterally, and potentially exfiltrate sensitive data.
- This campaign is part of a larger trend of targeted attacks against financial institutions, which often have access to large amounts of sensitive customer data.
