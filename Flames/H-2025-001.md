# H-2025-001

Threat actors are using PowerShell's Invoke-WebRequest cmdlet to download encrypted payloads from Discord CDN URLs to evade network detection.

## Why
- Malicious packages are abusing trusted developer tooling like npm and pip to deliver malware
- Encrypted payloads downloaded over trusted channels like Discord CDN can bypass network security monitoring
- Successful compromise gives attackers an initial foothold to deploy ransomware or establish persistence
