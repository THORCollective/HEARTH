# H-2025-001

In a recent campaign, threat actors have been observed using the BITSAdmin tool to download malicious payloads from compromised websites. BITSAdmin is a command-line tool that you can use to create download or upload jobs and monitor their progress. The threat actors are using this tool as it allows them to download files in the background, making it less likely to be detected by the user or security tools. The downloaded payloads are then used to compromise the system and steal sensitive data.

## Why
- Detecting the misuse of BITSAdmin is crucial as it is a legitimate Windows tool and its malicious use can easily blend with normal system activity, making it a popular technique for evading detection.
- If successful, the threat actors can download and execute malicious payloads, leading to system compromise and potential data theft.
- This behavior has been linked to larger campaigns where threat actors compromise websites to host their malicious payloads.
