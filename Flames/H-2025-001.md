# H-2025-001

Threat actors are using Chisel, an open-source tunneling utility, to create SOCKS proxies on compromised hosts in order to bypass network security controls such as firewalls and conceal C2 traffic.

## Why
- Tunneling allows adversaries to bypass perimeter defenses and network monitoring. Detecting tunnels is crucial to identify compromised hosts communicating with C2 servers.
- Chisel proxies enable the attacker to interact with multiple machines in the environment through a single compromised host, expanding access and evading detection.
- This tunneling technique is part of a larger campaign by an initial access broker targeting financial institutions across Africa. Identifying Chisel activity can reveal the scope of the intrusion.
