# H-2025-001

Threat actors are using the open-source Chisel utility to create tunnels that bypass network security controls and enable communication with compromised systems in victim environments.

## Why
- Chisel tunnels allow attackers to stealthily access compromised systems and move laterally within networks while evading firewall restrictions. 
- By establishing these covert communication channels, the threat actors can maintain persistent access to victim networks in the financial sector.
- This Chisel tunneling activity is part of a larger campaign by an initial access broker tracked as CL-CRI-1014 that is targeting financial institutions across Africa.
