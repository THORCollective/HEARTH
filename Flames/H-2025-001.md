# H-2025-001

Threat actors are using the open-source Chisel utility to create tunnels that proxy and forward network traffic through compromised victim machines to bypass firewalls and conceal C2 communications.

## Why
- Chisel tunneling enables adversaries to bypass network security controls like firewalls to maintain covert command and control channels. 
- Detecting Chisel abuse is important to identify compromised systems being used as proxies to hide the true source of malicious network traffic.
- This tunneling technique was a key part of the CL-CRI-1014 attack playbook targeting financial institutions across Africa, likely to establish initial access for sale on dark web markets.
