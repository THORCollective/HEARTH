# H-2025-001

Adversaries are modifying Windows Registry Run keys in HKCU\Software\Microsoft\Windows\CurrentVersion\Run to maintain persistence of malicious SimpleHelp services across system reboots.

## Why
- Attackers are exploiting known vulnerabilities in the SimpleHelp RMM software to gain initial access and remote control over endpoints
- By modifying Registry Run keys, adversaries can ensure their malicious SimpleHelp services automatically start at each reboot to maintain persistence
- This technique has been observed in recent campaigns deploying ransomware like DragonForce and Medusa after establishing access via compromised SimpleHelp instances
