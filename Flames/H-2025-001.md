# H-2025-001

Threat actors are using PowerShell's Invoke-WebRequest cmdlet to download encrypted KimJongRAT payloads from attacker-controlled CDN accounts to evade network detection.

## Why
- The KimJongRAT malware is a persistent and evolving threat that has been actively developed since 2019. Detecting its delivery is critical to prevent infections.
- Using legitimate CDN services to host malicious payloads allows threat actors to evade network security monitoring and blend in with normal traffic. 
- Identifying PowerShell downloading encrypted payloads from untrusted external sources can catch the initial access before later stages execute.
