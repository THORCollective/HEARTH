# H-2025-001

Threat actors are using Chisel, an open-source tunneling utility, to create SOCKS proxies on compromised hosts to bypass network security controls and conceal C2 traffic.

## Why
- Chisel proxies enable threat actors to hide their true origin and evade network detection
- Tunneling C2 traffic through proxies on infected hosts allows attackers to blend in with normal network activity
- Chisel has been used by the CL-CRI-1014 threat cluster to target financial institutions across Africa
