# H-2025-001

Threat actors are using the open-source Chisel utility to create tunnels that bypass network controls like firewalls and forward traffic from attacker-controlled servers to other remote machines in the victim environment.

## Why
- Chisel tunnels allow attackers to evade network security controls and access restricted systems
- Detecting this technique can disrupt lateral movement and command-and-control in the environment 
- The CL-CRI-1014 threat actor cluster has used Chisel tunneling to target financial institutions across Africa
