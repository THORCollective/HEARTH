# H-2025-001

Threat actors are using the ClickFix social engineering technique to trick users into copying and pasting malicious PowerShell commands into their operating system's run dialog, which results in the execution of the GHOSTPULSE loader and ultimately the deployment of the ARECHCLIENT2 information stealer on Windows systems.

## Why
- Detecting this behavior is crucial as it represents the initial access vector of a multi-stage attack that leads to the deployment of the GHOSTPULSE loader and the ARECHCLIENT2 information stealer. 
- If successful, the threat actors gain access to sensitive user data and system information, and also have remote control capabilities over the compromised systems. 
- This behavior is linked to a larger campaign that has seen a significant surge in activity throughout 2025, indicating that it is a preferred method of initial access for threat actors.
