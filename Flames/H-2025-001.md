# H-2025-001

Threat actors are using the ClickFix social engineering technique to trick users into executing a base64-encoded PowerShell command that, when decoded, downloads a malicious ZIP file from a remote server and extracts it into a temporary directory on the victim's system. This immediate action serves as an initial access vector for the deployment of the GHOSTPULSE loader and ultimately the ARECHCLIENT2 information stealer.

## Why
- Detecting this behavior is crucial as it allows threat actors to gain initial access to systems, which is the first step in a multi-stage attack. 
- If successful, this technique can lead to the deployment of the GHOSTPULSE loader and ultimately the ARECHCLIENT2 information stealer, which can result in substantial data loss and potential financial damage.
- This technique has been linked to a larger campaign that has seen a considerable increase in activity throughout 2025, indicating a potential ongoing threat.
