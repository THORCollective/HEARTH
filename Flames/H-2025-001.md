# H-2025-001

Threat actors are modifying the Windows Registry Run keys in HKCU\Software\Microsoft\Windows\CurrentVersion\Run to maintain persistence by executing malicious JavaScript code at user login that uses JSF*ck obfuscation composed primarily of the symbols []+${} to evade detection.

## Why
- Modifying Registry Run keys is a common persistence mechanism that ensures malicious code runs every time the user logs in
- JSF*ck obfuscation makes the malicious JavaScript code difficult to detect and analyze
- Persistence allows the attacker to maintain access even if their initial entry point is removed
- This TTP could enable long-term undetected access to the victim's system to steal data or deploy additional malware
