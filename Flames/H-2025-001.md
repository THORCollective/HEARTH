# H-2025-001

Threat actors are using Chisel, an open-source tunneling utility, to create SOCKS proxies on compromised machines in order to bypass network security controls and conceal C2 traffic.

## Why
- Chisel tunnels enable the attackers to bypass firewalls and network monitoring, allowing them to operate undetected in the victim environment.
- By proxying C2 traffic through the Chisel SOCKS proxy on infected hosts, the threat actors can interact with their tools like PoshC2 while evading network defenses. 
- This tunneling is a key part of the CL-CRI-1014 attack playbook targeting financial institutions across Africa. Detecting this technique can help identify this cluster of activity.
