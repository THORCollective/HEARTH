# H-2025-001

CTI Report:

Our intelligence sources have identified a new threat actor group that is leveraging the BITSAdmin tool to download malicious payloads from compromised websites. The BITSAdmin tool is a command-line tool that you can use to create download or upload jobs and monitor their progress. The threat actors are using this tool as it allows them to download files in the background, which can often bypass traditional network security measures. The downloaded payloads are then used to compromise the target systems and exfiltrate sensitive data.

---

Hypothesis:

Threat actors are using the BITSAdmin tool to download malicious payloads from compromised websites in the background to bypass network security measures and compromise target systems.

## Why
- Detecting the misuse of BITSAdmin tool is crucial as it is a legitimate Windows tool and its malicious use can easily blend with normal network traffic, making it difficult to detect.
- If successful, the threat actors can compromise the target systems and exfiltrate sensitive data, causing significant damage to the organization.
- This behavior has been linked to a new threat actor group, indicating a potential larger campaign.
