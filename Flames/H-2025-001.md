# H-2025-001

Threat actors are using the open-source tunneling utility Chisel to create SOCKS proxies on compromised systems, bypassing network controls like firewalls to conceal C2 traffic and pivot to other internal hosts.

## Why
- Tunneling tools like Chisel allow adversaries to hide their network traffic, blend in with normal activity, and evade perimeter security controls. This enables them to maintain covert communication channels and access other systems.
- Successfully detecting Chisel tunnels could identify compromised hosts being used as network proxies and uncover previously hidden lateral movement and C2 activity.
- The Chisel tool was used extensively by an initial access broker targeting financial institutions across Africa, as part of a broader campaign to establish footholds and sell network access.
