# H-2025-001

Threat actors are using Chisel, an open-source tunneling utility, to create SOCKS proxies on compromised hosts to bypass network security controls like firewalls and conceal C2 traffic.

## Why
- Chisel proxies allow adversaries to hide their true C2 infrastructure and evade network-based detection
- Tunneling C2 traffic through internal proxy hosts helps blend in with normal network activity
- Detecting Chisel usage can reveal compromised systems being used as C2 pivots in the environment
