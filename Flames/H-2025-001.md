# H-2025-001

Threat actors are using Chisel, an open-source tunneling utility, to create SOCKS proxies on compromised hosts to bypass network security controls like firewalls and conceal C2 traffic.

## Why
- Chisel proxies allow attackers to hide their true origin and evade network detection
- Tunneling C2 traffic through internal proxies lets adversaries operate covertly
- This technique was used by CL-CRI-1014 to compromise financial institutions in Africa
