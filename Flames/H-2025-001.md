# H-2025-001

Our threat intelligence team has recently observed a new campaign by an unknown threat group targeting financial institutions. The attackers are using spear-phishing emails to deliver a malicious Microsoft Word document. The document contains an embedded macro that, when enabled, executes a PowerShell command to download a second-stage payload from a remote server. The payload is a custom backdoor that communicates with the command and control (C2) server over HTTP and allows the attackers to execute arbitrary commands on the compromised system.

---

Hypothesis:

Threat actors are using malicious Microsoft Word documents with embedded macros to execute PowerShell commands that download a custom backdoor from a remote server, allowing them to execute arbitrary commands on compromised systems within financial institutions.

## Why
- Detecting this behavior is crucial as it allows threat actors to gain initial access to the target system and establish a foothold.
- If successful, the threat actors can maintain persistent access to the compromised system, potentially leading to data exfiltration, lateral movement, or further exploitation.
- This behavior is part of a larger campaign targeting financial institutions, which could have significant financial and reputational impacts.
