## Why

- UNC6201 created temporary vNICs on existing VMs — removed after use leaving minimal forensic evidence
- Most SOCs don't monitor ESXi host-level VM config changes
- Data sources: ESXi hostd.log, vpxa.log, vCenter task events for VirtualDevice.add
- TTP used since mid-2024 but only publicly reported Feb 2026
