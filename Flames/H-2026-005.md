# H-2026-005

Threat actors are exploiting file upload functionality in Cisco Catalyst SD-WAN Manager by uploading malicious CSV files containing embedded commands that append unauthorized root-privileged user accounts to /etc/passwd and /etc/shadow files to escalate from administrative to root-level access.

## Why
- This technique represents a critical privilege escalation vector that transforms administrative access into complete root control of SD-WAN infrastructure, enabling threat actors to bypass all security controls and maintain persistent access to network orchestration platforms
- Successful exploitation grants attackers root-level access to centralized SD-WAN controllers that manage entire network fabrics across distributed organizations, allowing interception and manipulation of all enterprise traffic flowing through branch locations
- This zero-day exploitation pattern (CVE-2026-20245) demonstrates sophisticated targeting of network edge devices that typically lack robust logging and forensic capabilities, making detection extremely difficult without proactive hunting for file upload anomalies and unauthorized account creation in system authentication files
- The technique is part of a broader "living off the edge" campaign where state-sponsored actors prioritize network appliance compromise to establish long-term strategic intelligence collection capabilities while evading traditional endpoint security solutions
