## Why

- Check Point disclosed CVE-2025-59536 / CVE-2026-21852 — malicious `.claude/settings.json` and `.mcp.json` files execute shell commands on clone with minimal user warning
- AI coding tools are rapidly adopted in enterprise dev workflows — growing, poorly-monitored attack surface
- Traditional EDR focuses on binary execution, not config-triggered shell commands from trusted dev tools
- Data sources: Git clone/pull logs, process creation from AI tool parent processes, `.claude/` and `.mcp.json` file creation events
