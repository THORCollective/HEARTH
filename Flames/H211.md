# H211

Threat actors are sending unauthenticated HTTP GET requests to the WordPress REST API endpoint /wp-json/gravitysmtp/v1/tests/mock-data with the query parameter ?page=gravitysmtp-settings to exfiltrate API keys, OAuth tokens, and detailed system configuration data from sites running vulnerable Gravity SMTP plugin versions.

## Why
- This vulnerability has been exploited at massive scale with over 17 million blocked attempts, indicating widespread active targeting by threat actors seeking to harvest credentials and reconnaissance data
- Successful exploitation provides attackers with live third-party email service credentials (Amazon SES, Google, Mailjet, Resend, Zoho) enabling email-based attacks and detailed system information for planning follow-on exploitation
- The trivial exploitation method (single unauthenticated GET request) combined with 100,000 active installations creates a significant attack surface, with peak exploitation reaching 4 million attempts in a single day on June 7th, 2026
- Detection of this specific REST API access pattern is critical as the vulnerability leaves minimal traces and many sites may remain unpatched, especially those using free versions of Wordfence that received protection 30 days after premium users
