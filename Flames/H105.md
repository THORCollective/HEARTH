## Why

- CVE-2023-46604 (CVSS 10.0) in Apache ActiveMQ has been widely exploited by ransomware operators including LockBit, and exploitation requires no authentication — a single crafted OpenWire command achieves remote code execution
- Internet-facing applications are the most common initial access vector in ransomware intrusions, and many organizations run vulnerable versions long after patches are available
- Exploitation of public-facing applications often leaves forensic artifacts in application logs, web server logs, and endpoint telemetry that differ from normal application behavior — such as Java processes spawning cmd.exe, PowerShell, or certutil
- Early detection of exploitation attempts at the initial access phase provides the best opportunity to contain an intrusion before lateral movement and ransomware deployment
