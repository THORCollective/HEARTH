# H081

An adversary is publishing typosquatted npm packages with MCP server injection targeting developer environments using AI coding assistants to harvest SSH keys, cloud credentials, and LLM API keys via prompt injection.

## Why

- MCP server injection is a novel attack vector — most security teams have zero visibility into AI tool configurations
- Worm propagates using stolen npm/GitHub tokens, meaning one compromised dev can seed packages across an org
- 48-hour delayed second stage with per-machine jitter evades sandbox analysis and incident response timelines
- Targets 9 LLM provider API keys — compromised keys enable downstream abuse at scale
