## Why

- BYOVD is increasingly common but many orgs lack vulnerable driver blocklists or kernel-level monitoring
- GPO abuse for ransomware distribution bypasses endpoint-focused detections — it looks like legitimate admin activity
- Multi-stage execution (disable defenses → in-memory payload) leaves minimal disk artifacts
- Weak AD credentials as initial access vector — huntable via authentication log baselines
