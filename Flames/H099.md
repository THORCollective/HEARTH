## Why

- Zero-click exploitation requires no macro execution, no "Enable Content" prompt — just opening a file is sufficient, eliminating the most common user-awareness defense
- AI agent execution chains introduce novel data access paths that traditional DLP and exfil detection rules were not designed to monitor
- Copilot Agent activity may be logged separately from traditional Office telemetry, creating blind spots in environments without unified O365 audit logging
- High-value spreadsheet data (financial models, PII, legal documents) is commonly stored locally or in OneDrive without additional access controls
- Detecting unusual Copilot Agent API calls or outbound data transfers correlated with Excel open events is a reliable hunt pivot
